The code behind [my personal site/blog backend strapi app](https://www.dimitristrechas.com).

# INSTALLATION NOTES

1. Clone repo `git clone https://github.com/dimitristrechas/personal-site-strapi.git`
2. Copy `.env.example` to `.env` and set the environment variables
3. `yarn` to install npm packages
4. `docker compose up --remove-orphans --build --force-recreate -d` to build the docker image
